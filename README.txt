Déploiement GitHub + Netlify :
1. Dézipper le dossier
2. Envoyer tout le contenu dans votre dépôt GitHub
3. Connecter le dépôt à Netlify
4. Aucun build spécial n'est nécessaire : site statique simple
5. Le formulaire de contact est prêt pour Netlify Forms

Important sur le partage :
- les boutons de partage fonctionnent de manière pertinente une fois le site publié en ligne
- en local, certains partages peuvent être limités ou moins propres à cause des liens file://

Ajustements simples possibles :
- modifier les textes dans index.html et publications.html
- ajuster les prestations et les livrables
- remplacer les PDF dans assets/pdf
- changer l'e-mail de contact si besoin
- compléter les mentions légales après immatriculation
